# Saving, Withdrawing and Ordering Protocol (swop)

Crypto trading wallet, implementing Swopblock's saving, withdrawing and ordering protocol that is deployed as a console app (CLI) for second layer processing just under the first mobile app (GUI) processing layer of the (1) mobile, (2) console, (3) application, (4) incentive, (5) consensus, (6) blockchain and (7) internet processing layers of Swopblock.
