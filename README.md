# Saving, Withdrawing and Ordering Protocol (swop)

Swop is the Swopblock crypto wallet implementing Swopblock's saving, withdrawing and ordering protocol that is deployed as the second processing layer console app (CLI) just under the first processing layer mobile app (GUI) of the (1) mobile, (2) console, (3) application, (4) incentive, (5) consensus, (6) internet and (7) blockchain processing layers of Swopblock.
